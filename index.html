Simon Game

Instructions

How are we from memory? Can you remember the color sequence? Dare yourself!

Simon is an educational game to test wit, reflexes and memory. A permanent random challenge, skill and observation.
Each move will show a new color that you will always have to repeat from the beginning in the same order, forming a color chain.
To do this you will click with the mouse on the appropriate color.
